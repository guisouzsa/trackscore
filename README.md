# Track Score

Idioma: Português | English

Este projeto é um sistema web simples para gerenciamento de avaliações pessoais de álbuns musicais. Ele permite ao usuário adicionar, visualizar, editar e excluir avaliações, além de marcar algumas como favoritas.

O objetivo é oferecer uma ferramenta prática para organizar opiniões sobre álbuns ouvidos, mantendo histórico de notas e favoritos de forma simples e intuitiva. O projeto foi desenvolvido como parte de um trabalho escolar.

---

## Funcionalidades

- Adicionar avaliações com nome do álbum, artista, imagem e opinião
- Editar avaliações já cadastradas
- Excluir avaliações existentes
- Visualizar todas as avaliações registradas
- Marcar até 4 avaliações como favoritas
- Filtrar e visualizar apenas as avaliações favoritas
- Sistema de login e logout com email e senha
